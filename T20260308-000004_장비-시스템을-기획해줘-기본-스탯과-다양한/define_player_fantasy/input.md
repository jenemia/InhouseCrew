# Task 입력

- Task ID: `define_player_fantasy`

## 사용자 요청

장비 시스템을 기획해줘. 기본 스탯과 다양한 옵션을 넣을 수 있어야해. 그리고 그 옵션을 변경하거나 강화할 수 있어 유저들이 파밍의 목적이 되도록 해야해. 장비의 수는 많지 않고 옵션으로 차별점을 줘야 하니까. 장비 옵션 만들기, 옵션 강화하기, 옵션 바꾸기 등의 시스템과 재료 아이템도 필요해. Data Model까지 계획.

## Task 설명

입력 요구사항 `{user_request}`와 앞선 게임 컨셉을 바탕으로 플레이어 정체성, 존재감, 감정 판타지를 정의한다.

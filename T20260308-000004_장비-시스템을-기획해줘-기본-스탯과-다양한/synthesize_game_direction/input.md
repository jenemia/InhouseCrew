# Task 입력

- Task ID: `synthesize_game_direction`

## 사용자 요청

장비 시스템을 기획해줘. 기본 스탯과 다양한 옵션을 넣을 수 있어야해. 그리고 그 옵션을 변경하거나 강화할 수 있어 유저들이 파밍의 목적이 되도록 해야해. 장비의 수는 많지 않고 옵션으로 차별점을 줘야 하니까. 장비 옵션 만들기, 옵션 강화하기, 옵션 바꾸기 등의 시스템과 재료 아이템도 필요해. Data Model까지 계획.

## Task 설명

입력 요구사항 `{user_request}`와 앞선 4개 결과를 종합해 팀 최종 결론을 만든다. 게임 한 줄 정의, 핵심 플레이어 판타지, unique hook, 핵심 시스템 방향성, 핵심 콘텐츠 방향성, 시장성 관점 보정 포인트, 다음 기획 단계 제안을 포함한다.
